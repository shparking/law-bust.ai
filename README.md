# Law-bust AI (로버스트 AI)

> **대한민국 법령 DB를 실시간 탐색해 실무 문서까지 자동 생성하는 AI 법무 보조 에이전트**
> legalize-kr를 기반으로 제작하였습니다.

| 항목 | 내용 |
|------|------|
| 에이전트 이름 | Law-bust AI (로버스트 AI) |
| 플랫폼 | Claude Cowork — claude.ai |
| 핵심 스킬 | `legalize-kr`, `docx` |
| 제작 배경 | 2026 SK 신입구성원 AI Agent 사전 과제 |

---

## 01 에이전트 소개

Law-bust AI는 대한민국 공개 법령 DB(법률·판례·행정규칙·조례)를 실시간으로 탐색해, 사용자의 구체적 상황에 적용되는 법 조항과 판례를 찾아주고 실무 제출 문서까지 자동으로 생성하는 AI 법무 보조 에이전트입니다.

별도 설치 없이 Claude Cowork(claude.ai)에서 상황을 자연어로 설명하면, 민법·근로기준법 등 현행 법률 전문과 대법원 판례 28,000여 건을 검색해 적용 근거를 제시하고, 내용증명·고소장·증거자료 양식을 Word 파일(.docx)로 즉시 출력합니다. 법률 지식이 없어도 권리를 지킬 수 있도록 설계했습니다.

```
사용자: "퇴직 통보했는데 회사가 막고 있어. 법적으로 대응하고 싶어."

Law-bust AI:
  1. legalize-kr 스킬 → 민법 §660, 근로기준법 §7·§20, 관련 대법원 판례 탐색
  2. docx 스킬       → 내용증명 / 고소장 / 증거자료 양식 .docx 3종 생성
  ✅ 완료 (약 15분)
```

- 법무법인 초기 상담료 10~30만원 이상 발생

- 상황 설명 후 **10분 내** 관련 법령·판례 및 대응 전략 제시

- 초기 법률 상담 비용 **건당 0원** (기존 대비 최대 30만원 절감)

## 03 실전 사례 — 퇴직 분쟁 대응

이 에이전트가 탄생한 실제 사례입니다.

**상황**: 삼성전자 재직 중 SK하이닉스 이직 결정 → 6월 12일 퇴직 통보(6/30 마지막 날) → 인사팀이 "1개월 전 통보" 사내 규정을 근거로 퇴직 거부

**Law-bust AI 워크플로우**:

```
1. legalize-kr 스킬로 법령 탐색
   ├─ 민법 제660조    — 고용 해지통고는 언제든지 가능
   ├─ 근로기준법 제7조 — 강제근로 금지 (위반 시 5년 이하 징역)
   └─ 근로기준법 제20조 — 위약예정 금지 (사내 규정 무효)

2. 대법원 판례 28,000건 중 관련 판례 검색
   └─ 대법원 2002다11458 등 의원면직 관련 판례 확인

3. docx 스킬로 문서 3종 자동 생성
   ├─ 내용증명_퇴직통보.docx
   ├─ 고소장_강제근로금지위반.docx
   └─ 증거자료_양식.docx
```

**소요 시간**: 약 15분 | **비용 절감**: 변호사 초기 상담료 10~30만원

### 적용 가능한 법률 사례 5가지

| # | 상황 | 탐색 법령 | 생성 문서 |
|---|------|----------|----------|
| 1 | **퇴직·이직 분쟁** — 회사가 퇴직 처리 거부 | 민법 §660, 근로기준법 §7·§20 | 내용증명, 고소장 |
| 2 | **임금 체불** — 급여·퇴직금 미지급 | 근로기준법 §43·§109, 임금채권보장법 | 고용노동부 진정서 |
| 3 | **부당해고** — 정당한 이유 없는 해고 통보 | 근로기준법 §23·§27 | 노동위원회 구제신청서 |
| 4 | **임대차 분쟁** — 보증금 반환 거부 | 주택임대차보호법 §3·§6, 상가임대차법 §10 | 내용증명, 지급명령 신청서 |
| 5 | **소비자 피해** — 환불 거부·허위 광고 | 소비자기본법 §16, 전자상거래법 §17 | 소비자원 피해구제 신청서 |

## 04 스킬 구조

이 에이전트는 Claude Cowork의 **스킬(Skill)** 시스템 위에서 동작합니다. 스킬은 Claude에게 특정 도메인 전문성을 부여하는 `SKILL.md` 지침 파일 패키지입니다.

```
claude-skills/
├── README.md
├── skills/
│   ├── legalize-kr/        ← 법령 탐색 스킬 (핵심)
│   │   ├── SKILL.md
│   │   └── scripts/
│   ├── docx/               ← Word 문서 생성 스킬 (핵심)
│   │   ├── SKILL.md
│   │   └── scripts/
│   ├── xlsx/               ← Excel 스프레드시트
│   ├── pptx/               ← PowerPoint 발표자료
│   ├── pdf/                ← PDF 처리
│   ├── deep-research/      ← 멀티소스 리서치
│   └── schedule/           ← 반복 작업 스케줄링
└── examples/
    ├── legal-notice/       ← 내용증명 생성 예제
    ├── research-report/    ← 딥리서치 리포트 예제
    └── data-analysis/      ← 엑셀 분석 예제
```

### 전체 스킬 목록

| 카테고리 | 스킬 | 설명 | 트리거 키워드 |
|---------|------|------|-------------|
| 법률·문서 | **legalize-kr** | 법률·판례·행정규칙·조례 탐색 및 조문 추출 | 법령 찾아줘, 판례 검색 |
| 법률·문서 | **docx** | Word 문서 생성·편집·분석 | 문서 만들어줘, .docx |
| 법률·문서 | **pdf** | PDF 생성·추출·병합·OCR | PDF, 변환, 합치기 |
| 데이터·생산성 | **xlsx** | Excel 생성·분석·수식·차트 | 엑셀, 표, 스프레드시트 |
| 데이터·생산성 | **pptx** | PowerPoint 생성·편집 | PPT, 슬라이드, 발표자료 |
| 데이터·생산성 | **deep-research** | 멀티소스 검색 → 팩트체크 → 인용 리포트 | 조사해줘, 리서치 |
| 자동화·시스템 | **schedule** | 반복 작업·리마인더 스케줄링 | 매일, 매주, n시간 후에 |
| 자동화·시스템 | **skill-creator** | 새 스킬 제작·개선·성능 평가 | 스킬 만들어줘 |

## 05 설치 방법

### 방법 1 — .skill 파일 (권장)

1. `skills/` 폴더에서 원하는 스킬의 `.skill` 파일 다운로드
2. Claude Cowork → **Settings > Capabilities > Skills** → 파일 드래그앤드롭

### 방법 2 — 플러그인 설정

```json
{
  "plugins": [
    {
      "name": "law-bust-ai",
      "skills": ["legalize-kr", "docx"]
    }
  ]
}
```

### 방법 3 — 수동 설치

```bash
git clone https://github.com/<your-org>/law-bust-ai.git
cp -r law-bust-ai/skills/legalize-kr ~/.claude/skills/
cp -r law-bust-ai/skills/docx ~/.claude/skills/
```

## 06 스킬 제작 가이드

새 스킬은 `skill-creator` 스킬로 만들거나, 아래 최소 구조를 따르세요.

```markdown
---
name: my-skill
description: 이 스킬을 언제 사용할지 한 문장으로. 트리거 키워드 포함.
---

# My Skill

## 언제 사용하나
## 워크플로우
## 도구 선택 기준
## 출력 형식
```

**좋은 스킬의 조건**: `description`에 트리거 키워드 명시 / 도구 선택 로직 명확 / 출력 형식 구체적 / **실제 작업을 10분 안에 완료할 수 있을 것**
